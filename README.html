PDF Password Protection Tool is a **command-line utility** for Windows that enables users to: ✅ Encrypt PDF documents with **military-grade AES-256** encryption ✅ Set **dual-password protection** (user & owner passwords) ✅ Apply **granular permissions** (printing, copying, editing) ✅ Maintain **data retention compliance** with 30-day expiration tracking ✅ Automate secure PDF workflows with **drag-and-drop** or batch processing **Use Cases:** - Legal document protection - Healthcare data compliance (HIPAA) - Financial records encryption - Confidential business communications - Academic research data security --- ## 🚀 Quick Start ### Drag & Drop Method (Recommended) ```bash # Simply drag your PDF file onto Protect-PDF.bat # Enter password when prompted # Encrypted file appears in OUTPUT/ folder ``` ### Command-Line Method ```bash # Double-click Protect-PDF.bat # Provide file path: C:\Documents\sensitive.pdf # Set password: ******** # Output: OUTPUT/sensitive_20251026.pdf ``` ### First-Time Setup ```powershell # Install Python (if not already installed) winget install Python.Python.3.12 # PyMuPDF library auto-installs on first run ``` --- ## Ensure no visual surveillance when entering credentials. ### Data Handling - **Original Files:** Remain unmodified - **Output Location:** `OUTPUT/` directory (auto-created) - **Naming Convention:** `{original_name}_{YYYYMMDD}.pdf` - **Collision Handling:** Appends `_1`, `_2`, etc. --- ## 📁 Project Structure ``` pdf-password/ │ ├── Protect-PDF.bat # Main executable ├── README.md # Documentation ├── README.html # Styled documentation │ ├── scripts/ │ ├── protect_pdf.py # Encryption engine │ └── Protect-PDF-Simple.ps1 # PowerShell wrapper │ └── OUTPUT/ # Encrypted PDFs (auto-created) ``` --- ## 🔍 Troubleshooting ### Common Issues
Error: "Python is not installed or not in PATH" **Solution:** ```powershell # Verify Python installation python --version # If not found, reinstall with PATH option checked winget install Python.Python.3.12 ```
Error: "PyMuPDF installation failed" **Solution:** ```powershell # Manual installation pip install --upgrade pip pip install PyMuPDF # Alternative: Use conda conda install -c conda-forge pymupdf ```
Script won't run - "execution policy" error **Solution:** ```powershell # Open PowerShell as Administrator Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser # Verify change Get-ExecutionPolicy ```
PDF output is corrupted **Solution:** - Ensure input PDF is not already encrypted - Verify sufficient disk space - Try re-running with a different output name - Check PyMuPDF version: `pip show PyMuPDF`
