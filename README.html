### 📦 First Run > 🎯 **Auto-Setup:** PyMuPDF library installs automatically on first run! --- ## 📖 How It Works ### 📁 Output File Format ``` Input: Document.pdf Output: OUTPUT/Document_20251024.pdf ``` **Multiple files same day:** ``` OUTPUT/Document_20251024.pdf OUTPUT/Document_20251024_1.pdf OUTPUT/Document_20251024_2.pdf ``` > 💡 **Note:** Original files remain untouched. New encrypted copies are created in **OUTPUT** folder. ### 🔐 Encryption Settings
Setting Details
🔒 Encryption AES-256 (Military-grade)
👤 User Password Opens & views PDF (high-quality printing)
👑 Owner Password Full access (copy, modify, annotate)
📜 Permissions High-quality printing for all users
### 🔑 Password Setup > ⚡ **Fast & Simple:** No password confirmation required! **Recommended strength:** - ✅ Minimum 8 characters - ✅ Uppercase + lowercase letters - ✅ Numbers + special characters ### 👤 User vs 👑 Owner Password | Password Type | Access Level | |---------------|--------------| | **Same password** (default) | Full access to view, print, copy, modify | | **Different passwords** | User: view-only • Owner: full editing rights | > ⚠️ **Owner Password:** Must be 12+ characters and different from user password ### 📊 Permission Comparison
| Action | 👤 User Password | 👑 Owner Password | |--------|:----------------:|:-----------------:| | Open & View PDF | ✅ | ✅ | | Print (High-Quality) | ✅ | ✅ | | Copy Text/Images | ❌ | ✅ | | Modify Content | ❌ | ✅ | | Add Annotations | ❌ | ✅ |

--- ## 🔍 Troubleshooting
❌ Error: "Python is not installed or not in PATH"
**Solution:** Install Python using the [Installation & Setup](#%EF%B8%8F-installation--setup) section above.
❌ Error: "PyMuPDF installation failed"
**Solution:** Manually install via PowerShell: ```powershell pip install PyMuPDF ```
❌ Script won't run - "running scripts is disabled"
**Solution:** Open PowerShell as Administrator: ```powershell Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser ```
❌ PDF file not found
**Solution:** - ✅ Verify file path is correct - ✅ Ensure file exists - ✅ Use drag-and-drop to avoid typos
